The program homework1 is a python program.
It can be run from the command line by using the command:-
python Homework1.py
or else it can be run using sublime text by using the command ctrl+b. 

It reads an input file which is the corpus.txt file.Also it reads an input file which is the sentence.txt file.It produces two output files, bigrams.txt and unigrams.txt.The bigrams.txt file contains the probabilities of the bigrams of the sentence .The unigrams.txt file contains the probabilities of the unigrams of the sentence.

The second part of the program populates the output files GoodturingforBigrams.txt and GoodturingforUnigrams.txt which contains smoothed unigram/bigram probabilities.
